API Key Anthropic
Per generare gli audit è necessaria una API Key Anthropic. Viene salvata solo nel suo browser.
Come ottenere la API Key
  1. Vai su console.anthropic.com
  2. Accedi o registrati
  3. Vai su Settings → API Keys
  4. Clicca "Create Key"
  5. Incolla la key qui sopra e salva
